안녕하세요 프론트엔드 개발자 신현균 입니다.
- 마음가짐 학부시절 웹 프로그래밍 수업을 통해 내가 작성한 코드로 실제 유저가 사용하는 웹 페이지를 만들 수 있다는것에 흥미를 가지고 개인적으로 웹 공부를 시작했습니다.
웹 프로젝트를 진행할 때 단순히 개발자 입장에서 구현하기보다 사용자의 입장에서 UX/UI를 고려하여 개발하는것이 중요하다고 생각합니다.
웹 공부를 처음 하게 됐을 때 기본기가 탄탄해야 급변하는 웹 트렌드에도 흔들리지 않을거라 생각해서 React, Vue같은 프레임워크를 먼저 공부하기보다 HTML, CSS, JS와 같은 기본기를 중점적으로 공부했습니다.
- 경험 간단한 프로젝트 구현을 하면서 필요한 기능을 구글링을 하는 방법과 구현 결과를 얻을 수 있었고, 인턴을 통해 개인 프로젝트가 아닌 배포되고있는 솔루션의 이슈사항을 직접 수정해보며 실무를 경험해봤습니다. 또한 크몽이라는 외주사이트를 통해 간단한 퍼블리싱 외주 작업을 맡으며 디자이너와 원활히 협업하는 방법을 체득했습니다.
